# Virtualr - VR Landing Page 🚀

Virtualr is a modern and responsive landing page designed for Virtual Reality products and services. This project is built using React and Vite, focusing on clean UI design, component-based architecture, and performance optimization.

## 🔥 Features

- Responsive design (Mobile + Desktop)
- Modern UI/UX for VR products
- Reusable React components
- Fast performance with Vite
- Clean and structured code

## 🛠️ Tech Stack

- React.js
- Vite
- Tailwind CSS
- JavaScript (ES6+)

## 📂 Project Structure

- Components-based architecture
- Organized folders for scalability
